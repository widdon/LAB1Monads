# Система управления парковкой

> Консольная симуляция работы платной парковки с почасовой тарификацией,  
> штрафами за потерю билета, накоплением выручки и подробным логированием.  
> Реализована на Scala с использованием функциональных эффектов (`Reader`, `State`, `Writer`, `IO`).

## Особенности

- **Въезд автомобиля** – автоматическое назначение свободного места, запись времени въезда.
- **Выезд** – расчёт стоимости по тарифу и правилу округления времени (вверх / не округлять).
- **Потеря билета** – фиксированный штраф, машина покидает парковку.
- **Управление временем** – ручное переключение часов (`next`).
- **Финансовый учёт** – накопление выручки от всех операций.
- **Детальные логи** – каждое действие записывается с пояснением.
- **Гибкие настройки** – вместимость, тариф, штраф, режим округления задаются в одном месте.
